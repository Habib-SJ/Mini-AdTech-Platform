# Mini-AdTech-Platform
A simplified click-based advertising platform built to explore AdTech system design.


This project is a simplified click-based advertising platform built to explore the challenges of AdTech system design, including ad serving, campaign management, impression and click tracking, CTR analytics, dashboards, and reporting.

Project Phase Breakdown:
  [] - Phase 0: Repository & Project Setup
  [] - Phase 1: Models & Business Logic (User, Campaign, Ad, Impression, Click) + CTR/CPC Calculation
  [] - Phase 2: Statistics Dashboard, Daily Reports, Top Campaigns (Aggregation Queries)
  [] - Phase 3: Django REST Framework + API Documentation
  [] - Phase 4: Redis Cache + Docker (If time permits)
  [] - Phase 5: Unit Testing
